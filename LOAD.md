# CDP Helper Extension - Load Instructions

## Quick Start

### Method 1: Manual Load (Recommended)

1. Open Chrome / Edge and visit: `chrome://extensions/`
2. Enable **Developer mode** (toggle in top-right corner)
3. Click **"Load unpacked"**
4. Select this folder:  
   `/Users/onezion12344/WorkBuddy/20260428152655/cdp-helper-extension/`
5. Done! Click the extension icon in your toolbar to use it.

---

### Method 2: Auto-Load with Script

```bash
# For Chrome
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
  --load-extension=/Users/onezion12344/WorkBuddy/20260428152655/cdp-helper-extension

# For Edge
"/Applications/Microsoft Edge.app/Contents/MacOS/Microsoft Edge" \
  --load-extension=/Users/onezion12344/WorkBuddy/20260428152655/cdp-helper-extension
```

---

## After Loading

1. Click the **CDP Helper** icon in your toolbar
2. Select your **OS** (macOS / Windows / Linux)
3. Select your **Browser** (Chrome / Edge / Chromium / Brave)
4. **Copy** the startup command
5. Paste and run it in **Terminal**
6. Click **"Recheck CDP Status"** — should turn green!
